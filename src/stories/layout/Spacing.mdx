# Spacing & Padding Tokens

We strictly adhere to a **4px base grid layout** for all margin, padding, and gap properties.

## The Scale
All spacing classes in Tailwind map directly to this 4px multiplier:
- `0.5` : 2px (subtle interior gap)
- `1` : 4px (tight element spacing)
- `2` : 8px (base inner padding)
- `4` : 16px (standard container padding / gaps)
- `6` : 24px (relaxed container gaps)
- `8` : 32px (large component separation)
- `12` : 48px (section spacing)
- `16` : 64px (major section break)
- `24` : 96px (hero section breathing room)

*Rule*: Do NOT use arbitrary bracket spacing like `p-[17px]` or `gap-[21px]`. Always snap to the nearest token on the 4px grid.
