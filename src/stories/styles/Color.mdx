# Color System

Our color system is built entirely on OKLCH, powered by a single `--brand-hue` CSS variable.

## Brand Colors
<ColorPalette>

    title="Primary"

  <ColorItem

  <ColorItem
    title="Muted & Border"

</ColorPalette>

## Usage
Always use the Tailwind semantic classes (`bg-primary`, `text-muted-foreground`), never hardcoded Hex codes.
