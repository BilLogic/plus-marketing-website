# Icons

We standardize on **Lucide React**.

## Standards
- **Wait/Import Strategy**: Always import explicitly from `lucide-react` (e.g., `import { ChevronRight } from "lucide-react"`).
- **Inline Text**: Use `w-5 h-5`.
- **Feature Callouts**: Use `w-8 h-8` with a primary color tint (`text-primary`).

Do not mix icon libraries.
