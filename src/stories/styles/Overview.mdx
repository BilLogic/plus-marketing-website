# Styles Overview

Welcome to the **Plus Marketing Design System**. This documentation serves as the single source of truth for all styling tokens, replacing static Figma documentation.

## Core Philosophy
We build using a "lazy-genius" OKLCH dynamic color token approach, adhering strictly to a 4px layout grid, and embracing premium framer-motion micro-interactions.

## Categories
- **Color**: Semantic OKLCH tokens based on `--brand-hue`
- **Typography**: Display and Body text scales
- **Elevation**: Shadows, depth, and glassmorphism standards
- **Icons**: Standardized Lucide React implementation
- **Motion**: Cinematic and subtle interaction baselines
- **Shape**: Border radius and container geometries
